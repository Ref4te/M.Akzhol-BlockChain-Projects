## ⚠️ Безопасность и запуск
Все приватные ключи и RPC-адреса хранятся в файлах **`.env`**, которые **исключены** из репозитория.

**Порядок запуска:**
1. Перейти в папку проекта: `cd project1`
2. Установить зависимости: `npm install`
3. Создать файл `.env` (см. инструкцию внутри папки).
4. Запустить скрипт: `node deploy.js`
