## 1. Подготовка окружения

Откройте терминал и перейдите в директорию проекта:

cd projects/project1
npm install

Команда npm install устанавливает необходимые зависимости проекта
(ethers, dotenv) в папку node_modules.

## 2. Запуск локальной сети (Anvil)

Откройте второе окно терминала и запустите локальную сеть Ethereum:

anvil

После запуска Anvil в терминале появятся тестовые аккаунты, Private Keys и RPC URL.
⚠️ Это окно должно оставаться открытым, так как Anvil запускает локальный блокчейн.

## 3. Настройка переменных окружения (.env)

Вернитесь в первое окно терминала и создайте файл .env:

touch .env // или через VS code проводник

Откройте файл и добавьте данные из Anvil:

PRIVATE_KEY=0x_YOUR_PRIVATE_KEY_FROM_ANVIL
RPC_URL= "http://127.0.0.1:8545"

Или RPC тестовой сети и PRIVATE_KEY от кошелька MetaMask
Файл .env используется для хранения приватного ключа и адреса RPC.

## 4. Компиляция смарт-контракта

Выполните компиляцию контракта:

solcjs --bin --abi --include-path node_modules/ --base-path . Note.sol

После выполнения команды будут созданы файлы:

.abi — интерфейс контракта для JavaScript
.bin — байткод для загрузки в блокчейн

## 5. Деплой контракта

Запустите скрипт деплоя:

node deploy.js

Скрипт подключится к локальной сети Anvil, подпишет транзакцию вашим Private Key и отправит контракт в блокчейн. После успешного деплоя в терминале появится адрес задеплоенного смарт-контракта.
